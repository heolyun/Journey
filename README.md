# Journey Portfolio

웹 개발에서 인프라 엔지니어로 성장해 온 흐름을 보여주는 포트폴리오입니다.
프로젝트 목록보다 성장 과정, 운영 관점, 클라우드 확장 경험이 먼저 보이도록 구성합니다.

## Stack

- React
- TypeScript
- Tailwind CSS
- Vite

## Run

```bash
npm install
npm run dev
```

## Structure

```text
src/
  components/      UI sections
  data/            portfolio content and timeline data
  App.tsx          section composition
  styles.css       Tailwind entry and shared utilities
public/
  images/          project visual assets
```

## Content Direction

- Hero
- Featured Projects
- About Me
- Growth Journey
- Future Roadmap

Burinake를 가장 크게 강조하고, Growth Journey는 프로젝트와 트러블슈팅을 하나의 성장 타임라인으로 통합합니다.
